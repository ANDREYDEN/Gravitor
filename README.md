# Gravitor
Gravitor is a simple multilevel platformer made with Unity. 

## Gallery

![Sample](.\Screenshots\Sample.png)

## Instructions

To launch the game go to .\\Releases\\\<choose version\>\\Gravitor.exe

#### Controls:

Movement - WASD or arrow keys 

Jumping - Space

Pause - Esc
